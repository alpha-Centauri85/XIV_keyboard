# XIV_keyboard

A Dalamud plugin that provides an on-screen keyboard designed for controller first input in Final Fantasy XIV.

The goal is to make text input and key selection practical and comfortable without relying on a physical keyboard.

## Features
- On-screen keyboard UI
- Controller-based navigation (D-pad)
- Designed to complement console style play
- Lightweight and non intrusive

## Installation
1. Install Dalamud.
2. Add the plugin repository URL (once published).
3. Enable **XIV_keyboard** from the plugin list.

> Note: This plugin is under active development.

## Usage
- Open the keyboard via the configured command or keybind
- Navigate keys using your controller
- Confirm to send input to the game
